# 筋トレ記録共有アプリ

友達と一緒に筋トレを頑張るため、筋トレの実績を記録し、コメントやいいね！で励まし合えるアプリです。

## プロジェクト構成

```
muscle-app/
├── frontend/     # React フロントエンド
├── backend/      # NestJS バックエンド
├── docker/       # Docker設定ファイル
└── CLAUDE.md     # 開発仕様書
```

## 技術スタック

### フロントエンド
- React 18+
- TypeScript
- Material-UI / Chakra UI

### バックエンド
- NestJS
- TypeScript
- Prisma ORM

### データベース
- PostgreSQL 15+
